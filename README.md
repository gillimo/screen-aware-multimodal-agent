# AgentOSRS

Local-first OSRS decision agent project.

Quickstart:
- CLI: `python run_app.py status`
- Plan: `python run_app.py plan`
- GUI: `python run_app.py gui`

OSRS Client Overlay (planned):
- Primary surface is a silent overlay attached to the OSRS original client.
- Until the overlay ships, use the Tk GUI for planning and control.

Edit `data/state.json` to match your account.
