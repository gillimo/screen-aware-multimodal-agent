# Metrics and Ratings

All ratings are 0-100 and computed from local data.

Core metrics:
- Combat Readiness: defense + offense + survivability + mobility.
- Quest Readiness: prereqs met and quest unlock depth.
- Skilling Readiness: skill baselines and tool unlocks.
- Money Pathing: current cashflow vs target goals.
- Unlock Density: number of unlocks gained per hour.

Secondary metrics:
- Risk rating: gear risk and death cost sensitivity.
- Efficiency rating: xp per hour and gp per hour vs target.
- Bottleneck score: top blockers to desired content.
- Human-likeness: timing, motion, error rates, and session rhythm vs baselines.

Scoring notes:
- Ratings should explain which data points caused changes.
- All scores must show at least 3 concrete reasons.
