# Human-Likeness KPI Output

Purpose: define the output format for human-likeness scoring.

## Output Fields
- total_events: number
- unique_action_types: number
- reaction_ms_avg: number
- notes: string

## Example
```json
{
  "total_events": 120,
  "unique_action_types": 6,
  "reaction_ms_avg": 240.5,
  "notes": "stub scorer; replace with calibrated metrics"
}
```
