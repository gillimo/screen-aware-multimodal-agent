# Scope Boundaries

## v1
- Local JSON state only.
- Basic ratings with reasons.
- Plan generation with short/mid/long horizons.
- OSRS original client overlay as primary surface.
- Tkinter companion UI for dashboards and notes.

## v2
- Full OSRS reference datasets.
- Local model integration for narrative coaching.
- Expanded readiness checks and gear optimization.

## v3
- Screen-aware perception with structured state + visual cues.
- Optional action execution with human-like input behavior.
- Human-likeness metrics, calibration, and evaluation tooling.
